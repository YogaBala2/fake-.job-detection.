Fake Job Detection System
## Project Overview
       This project is a Flask-based web application that detects fraudulent job postings using machine learning techniques.

##  Features

* Detects fake job postings
* User-friendly web interface
* Machine learning model integration

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* Pandas
* HTML/CSS

## ▶️ How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Run the app:

   ```
   python app.py
   ```
3. Open browser:

   ```
   http://127.0.0.1:5000
   ```
